# The "Drunk Thumb" Protocol (UI/UX)

All frontend work (React/Tailwind) must adhere to:
1. **Mobile-First**: Design for 375px width by default.
2. **High Contrast**: Dark mode default (Slate-900/Black).
3. **Large Targets**: Minimum touch target size is 44px.
4. **Scannability**: Use `tracking-widest` for headers and clear visual hierarchy.
