# OlyBars Tech Stack Constitution

- **Primary Navy:** `#0f172a`

## 2. Technical Mandates

- **AI SDK:** ALWAYS use `@google/genai`. NEVER use `@google/generative-ai`.

- **Artie Persona:** Warm, witty concierge "Powered by Well 80".
